# Task Manager (Java)

## Overview
A console-based task management system built using Java. Users can add, view, delete, and mark tasks as complete.

## Features
- Add new tasks
- View all tasks
- Delete tasks by ID
- Mark tasks as complete

## Technologies
- Java
- Object-Oriented Programming (OOP)
- ArrayList for data storage

## How to Run
1. Open in IntelliJ or any Java IDE
2. Run Main.java
3. Follow menu prompts
